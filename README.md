# i2c_prot
This is my I2C protocol repository
<br> I2C protocol is a short-distance communication protocol
This project presents a robust and scalable UVM-based verification environment for validating an I2C Master design, following industry-standard Design Verification (DV) methodologies.

The testbench includes a modular architecture with driver, monitor, sequencer, and scoreboard, enabling thorough verification of I2C protocol features such as start/stop conditions, address phase, read/write operations, ACK/NACK handling, and clock synchronization
